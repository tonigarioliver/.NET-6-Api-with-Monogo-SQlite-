# LearnApi
This is a .NET 6 RestApi which covers the basics fucntionalities of:
  -CRUD to:
      -SQlite db
      -MonogDb db
  -JWT authentication:
      -Regsiter
      -Login
      -Refresh token
  - File handling:
      -Images
      -Excel Files
The full application implements the repository pattern and the unit of work to work with crud data
